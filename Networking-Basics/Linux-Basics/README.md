* `mkdir` / `touch`: Creating directories and files.
* `cp` / `mv` / `rm`: Copying, moving, and removing data.

## 🔐 Permissions & Security (chmod)
A critical skill for any security professional is managing file permissions.
I have practiced using the **Numeric Method** for `chmod`:

* **Read (4), Write (2), Execute (1)**
* **Example:** `chmod 755 filename`
  * **7 (4+2+1):** Full access for the owner.
  * **5 (4+1):** Read and execute for group and others.
